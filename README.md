# Car Price Prediction using Machine Learning

## CodeAlpha Data Science Internship

### Project Overview

This project aims to predict the selling price of used cars using Machine Learning techniques. The model is trained on historical car data and learns the relationship between various vehicle characteristics and their market value.

### Dataset Features

The dataset contains the following attributes:

* Car_Name
* Year
* Selling_Price (Target Variable)
* Present_Price
* Driven_kms
* Fuel_Type
* Selling_type
* Transmission
* Owner

### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook

### Machine Learning Algorithm

* Random Forest Regressor

### Project Workflow

1. Data Collection
2. Data Cleaning and Preprocessing
3. Feature Engineering
4. Label Encoding of Categorical Variables
5. Train-Test Split
6. Model Training
7. Price Prediction
8. Model Evaluation
9. Data Visualization

### Evaluation Metrics

The model performance was evaluated using:

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* R² Score

### Visualizations

The project includes:

* Actual vs Predicted Car Price Graph
* Feature Importance Analysis
* Model Performance Results

### Results

The Random Forest Regression model achieved strong prediction accuracy and successfully estimated car selling prices based on vehicle characteristics.

### Real-World Applications

* Used Car Marketplaces
* Vehicle Valuation Systems

* Insurance Price Assessment
